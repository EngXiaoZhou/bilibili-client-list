# Bilibili Client List

> 一个收集主流第三方哔哩哔哩APP开源/闭源项目的仓库。

---

##  前言

随着"**降本增笑**"的趋势来临，哔哩哔哩官方客户端随着版本更新也越来越臃肿，包括但不限于：

- 更多的广告
- 更臃肿无用的功能
- 植入API的PCDN播送，并且不优先采用CDN（**国际版官端也无法规避**）

这些缺陷已经大幅影响用户体验，因此社区出现了许许多多的 **第三方客户端 / 重实现项目**，它们大部分都可以解决上述痛点。

本仓库旨在**中立地整理与索引这些项目**，方便开发者与用户查阅、对比和研究。

> 说明  
> - 本仓库**不提供任何二进制文件**，仅收集项目链接与基本信息  
> - 所有项目版权归原作者所有  
> - 第三方客户端可能依赖官方 API，存在随时失效的可能  
> - 使用相关项目请自行评估合规性与风险
